WHAT IS THIS FOLDER?
====================

Shrav — AI & Automation Services website.

A personal landing site advertising your AI and automation services for
small businesses (dark, futuristic styling). Built to send to friends and
friends-of-friends.

Tech: React + Vite + Tailwind CSS + Framer Motion. Deployed on Vercel.

This is the ACTIVE, up-to-date copy of the site.
Git branch: claude/shrav-ai-services-site-r0EOO
GitHub repo: github.com/ShravG1/LUN

To open it: open this folder in VS Code (File > Open Folder).
To run it locally: open Terminal here and run  npm install  then  npm run dev

NOTE: There is a second, older clone of the same GitHub repo at /Users/shravg/LUN.
That one is NOT the live site — this folder is. See the note in ~/LUN.
